## 🗺️ 旺山经典徒步路线

### 轻松路线（约2小时）

```
百万叔停车场 → 茶园漫步（30min）
           → 网红旺字打卡（15min）
           → 林荫小路（30min）
           → 宝华寺祈福（30min）
           → 返回百万叔午餐
```

### 进阶路线（约3-4小时）

```
百万叔停车场 → 旺山登山道（1.5h）
           → 山顶观景台（30min）
           → 宝华寺（30min）
           → 茶园环线（40min）
           → 返回百万叔午餐
```

## 💡 徒步贴士

1. **早出发**：8:30前到，上午凉爽适合徒步
2. **轻装上阵**：车停百万叔，只带水和手机
3. **先点菜**：出发前可以先去店里看一眼菜单，回来直接点
4. **喝鸡汤**：秋冬徒步后来碗肚包鸡汤，暖身又滋补
5. **宝华寺**：免费开放，香火旺，值得一拜
